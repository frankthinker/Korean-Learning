# 韩语多级别语法学习网页

一款功能完善、交互友好的韩语语法学习平台，专为不同阶段的学习者设计。

## 🌟 核心特性

### 📚 完整的语法知识体系
- **初级语法**：涵盖终结词尾、助词用法、基本时态、简单连接词等入门必备知识
- **中级语法**：进阶语法如使动/被动语态、间接引语、惯用型、让步/条件连接词等
- **高级语法**：复杂语法、敬语体系深化、文学性语法、惯用俗语等

### 🎯 强大的功能模块

#### 语法检索与导航
- 分级导航：轻松切换初级、中级、高级内容
- 分类筛选：按词性、功能快速查找语法点
- 关键词搜索：支持韩文形式和中文释义双重搜索

#### 学习与练习
- 详情页展示：完整的定义、例句、注意事项、拓展词汇
- 即时练习：选择题、填空题、翻译题等多种题型
- 答题评分：实时反馈，自动记录错题

#### 个性化学习
- 📌 **我的收藏**：标记重点语法点，集中管理
- ✏️ **错题集**：自动记录错题，支持重新作答
- 📊 **学习进度**：实时显示学习完成度
- 💾 **本地存储**：所有数据自动保存，无需登录

#### 交互体验优化
- 🌙 深色模式：保护视力，适应不同使用环境
- 👨‍🏫 操作引导：首次使用自动显示操作提示
- 📱 响应式设计：完美适配手机、平板、桌面

### 📖 学习资源
- **语法技巧**：针对各级别的学习方法建议
- **拓展词汇**：常用搭配词汇，附词性和释义
- **文化背景**：语法的文化含义和使用场景

## 🚀 快速开始

### 安装依赖
```bash
npm install
```

### 开发模式（推荐方式）
```bash
python scripts/launch_dev.py
```
脚本将自动：
- 启动开发服务器（端口5176）
- 自动打开浏览器

### 开发模式（传统方式）
```bash
npm run dev
```
访问 `http://localhost:5176` 开始使用（注意：如果5176端口被占用，Vite会自动尝试其他端口，如5177、5178等，实际使用的端口会在终端中显示）

### 生产构建
```bash
npm run build
```

### 预览构建结果
```bash
npm run preview
```

## 🎨 设计特点

### 颜色主题
- **主色调**：蓝色系（#4a90e2）
- **辅助色**：浅灰色（#f5f5f5）
- **强调色**：红色（#ff6b6b）
- 支持深色模式切换

### 响应式布局
- 桌面版：三栏布局（导航 + 侧边栏 + 主内容）
- 平板版：自适应栅格
- 手机版：移动优化，侧边栏折叠

## 💾 数据存储

所有用户数据通过浏览器本地存储管理：
- **收藏夹**：favorite.js API
- **错题集**：wrong_answers.js API
- **学习进度**：learning_progress.js API
- **用户设置**：dark_mode 等个人偏好

## 🔧 技术栈

- **前端框架**：React 18
- **构建工具**：Vite
- **样式**：CSS3 (CSS Variables, Grid, Flexbox)
- **状态管理**：React Hooks
- **存储**：LocalStorage API

## 🎓 学习方式建议

### 初级学习者
- 多背诵例句，建立语感
- 按功能分类学习词尾
- 频繁复习巩固基础
- 尝试造句练习

### 中级学习者
- 对比学习易混淆语法
- 关注语法搭配和用法
- 阅读原文材料
- 学习剧集台词

### 高级学习者
- 理解文化背景
- 阅读文学作品
- 学习演讲表达
- 分析复杂句式结构
