# 快速使用指南

## 🎯 首次启动

### 1. 进入项目目录
```bash
cd /Users/shunhu/Documents/Codes/korean-grammar-learning
```

### 2. 启动开发服务器
```bash
npm run dev
```

### 3. 在浏览器中访问
打开浏览器访问 `http://localhost:5173`

## 📖 使用步骤

### 第一次使用
- 应用会自动显示欢迎提示
- 点击"开始学习"关闭提示

### 浏览语法
1. **选择级别**：顶部导航栏选择 初级/中级/高级
2. **分类筛选**：左侧边栏按词性或功能分类筛选
3. **搜索**：顶部搜索框输入韩文或中文搜索

### 学习语法
1. 点击语法卡片进入详情页
2. 查看完整的定义、例句和注意事项
3. 点击"拓展词汇"了解相关词汇
4. 点击"练习"标签页进行练习

### 练习题目
- **选择题**：点击选项后点击"提交答案"
- **填空题**：输入答案后点击"提交答案"
- **翻译题**：输入韩文后点击"提交答案"
- 查看解析和正确答案后点击"下一题"

### 管理收藏
- 在语法卡片或详情页点击❤️图标收藏
- 在"我的收藏"板块查看所有收藏的语法

### 查看错题
- 做错的题目自动保存在"错题集"
- 可以查看错误答案和解析
- 点击"重新练习"回到该语法的练习页面

### 学习进度
- 点击"标记为已掌握"记录学习进度
- 浏览页面的进度条实时显示学习完成度

### 深色模式
- 点击顶部右边的 🌙 图标切换深色/浅色模式
- 设置自动保存

## 🎓 学习建议

### 初级阶段
- 重点：词尾和助词
- 方法：多背诵例句，反复练习
- 目标：掌握基础语法，能理解简单对话

### 中级阶段
- 重点：复杂语法关系，易混淆形式对比
- 方法：学习原文材料，对比学习
- 目标：理解中等复杂句子，能进行基本对话

### 高级阶段
- 重点：文学语法，敬语系统
- 方法：阅读原文，理解文化背景
- 目标：掌握复杂表达，进行深入交流

## 🐛 常见问题

### Q: 如何清除所有数据？
A: 打开浏览器开发者工具 > Application > LocalStorage > 选择本站点 > 删除所有条目

### Q: 为什么看不到我之前的收藏？
A: 确保浏览器允许本地存储，检查隐私/无痕模式是否开启

### Q: 如何添加新的语法点？
A: 编辑 `src/data/grammarData.js` 文件，添加新语法数据

### Q: 如何自定义配色？
A: 编辑 `src/styles/global.css` 文件中的 CSS 变量 `:root` 部分

## 📱 响应式设计
- **桌面**（>1024px）：完整三栏布局
- **平板**（768px-1024px）：自适应栅格
- **手机**（<768px）：单栏布局，侧边栏可折叠

## 🎨 自定义主题

编辑 `src/styles/global.css` 中的变量：

```css
:root {
  --primary-color: #4a90e2;      /* 主色 */
  --secondary-color: #f5f5f5;    /* 辅助色 */
  --accent-color: #ff6b6b;       /* 强调色 */
  --success-color: #2ecc71;      /* 成功色 */
  --warning-color: #f39c12;      /* 警告色 */
  --error-color: #e74c3c;        /* 错误色 */
}
```
